# 🧠 BrainSpark

BrainSpark is an interactive brain-training web platform designed to improve cognitive skills through engaging games and quizzes.

## 🚀 Features
- User registration and login
- Quiz and puzzle games
- Score tracking system
- Admin dashboard
- Responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
